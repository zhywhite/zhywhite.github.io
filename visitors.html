Current visitor lookup

This lookup runs only after the button is clicked. It uses ipapi.co to estimate the public IP location for the current browser session.

How to enable full visitor statistics

To see all global clicks over time, connect a real analytics service or a small serverless endpoint. The recommended public display is aggregated by country, region, or city instead of exposing exact IP addresses.

Good options for GitHub Pages are Cloudflare Web Analytics, GoatCounter, Plausible, Google Analytics, or a self-hosted API that writes anonymized visit events.

Why not list raw IPs?

IP addresses can identify or track people across visits. Publishing them publicly is privacy-sensitive, and many analytics platforms intentionally avoid showing raw visitor IPs for that reason.

If you want an owner-only dashboard later, this page can link to it after the service is configured.
